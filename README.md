# Restaurant WhatsApp Bot

This is a simple Whatsapp chatbot for restaurants.

### Features 

- Auto reply on Whatsapp
- Menu with prices
- veg / Non veg / Drinks / Desserts
- Timing and location response
- Customer phone number auto save
- Repeat customer detection
- Monthly customer tracking 
- Popular dishes tracking 
- Data stored automatically
 
 # Tech Used

- python 
- Flask
- Twilio WhatSApp API

This project is created as a demo for client presentation.
